Beautiful Soup库的引用

Beautiful Soup库，也叫Beautiful Soup4或bs4
一般引用方式为

from bs4 import BeautifulSoup #从bs4中，引入BeautifulSoup这个类

也可以直接引用

import bs4

Beautiful Soup库基础理解

以HTML库页面为例。
Beautiful Soup库是解析、遍历、维护”标签树“的功能库，可以解析以标签树为基本格式的文档。

可以认为HTML文档<=>标签树<=>BeautifulSoup类，互相等价，BeautifulSoup对应一个HTML/XML文档的全部内容。
在此基础上，可以使标签树成为一个变量，对其BeautifulSoup类的改变，可等价于改变标签树

Beautiful Soup库解析器

都可以有效解析HTML/XMl文档，使用不同解析器会产生些微差距，需根据实际情况选择。
可供参考
https://beautifulsoup.readthedocs.io/zh_CN/v4.4.0/#id53

soup = BeautifulSoup('<html>data</html>'，'解析器种类')
解析器 使用方法 优势 劣势
Python标准库 BeautifulSoup(markup, “html.parser”) Python的内置标准库，执行速度适中 Python 2.7.3 or (3.2.2)前 的版本中文档容错能力差
lxml HTML 解析器 BeautifulSoup(markup, “lxml”) 速度快 ，文档容错能力强 需要安装C语言库
lxml XML解析器 BeautifulSoup(markup, [“lxml”, “xml”])、BeautifulSoup(markup, “xml”) 速度快 ，唯一支持XML的解析器 需要安装C语言库
html5lib BeautifulSoup(markup, “html5lib”) 最好的容错性，以浏览器的方式解析文档，生成HTML5格式的文档 速度慢，不依赖外部扩展

BeautifulSoup类的基本元素

<p>...</p>为标签
p为名称，
class=”title”为属性，用于定义标签特点，
class属性内容为”title”，属性可以说由键值对构成，class为键，title为值。

基本元素说明

<p class=“title”> … </p>'

Tag 标签，最基本的信息组织单元，分别用<>和</>标明开头和结尾
Name 标签的名字，"<p>…</p>"的名字是’p’，格式：.name
Attributes 标签的属性，字典形式组织，格式：.attrs
NavigableString 标签内非属性字符串，<>…</>中字符串，格式：.string
Comment 标签内字符串的注释部分，一种特殊的Comment类型

demo页面示例：

以下示例均基于http://python123.io/ws/demo.html网站

>>> import requests
>>> from bs4 import BeautifulSoup
>>> r = requests.get("http://python123.io/ws/demo.html")
>>> demo = r.text
>>> demo
'<html><head><title>This is a python demo page</title></head>\r\n<body>\r\n<p class="title"><b>The demo python introduces several python courses.</b></p>\r\n<p class="course">Python is a wonderful general-purpose programming language. You can learn Python from novice to professional by tracking the following courses:\r\n<a href="http://www.icourse163.org/course/BIT-268001" class="py1" id="link1">Basic Python</a> and <a href="http://www.icourse163.org/course/BIT-1001870001" class="py2" id="link2">Advanced Python</a>.</p>\r\n</body></html>'
>>> soup = BeautifulSoup(demo,"html.parser") #将demo做成汤

Tag 标签

>>> soup.title #soup标签
<title>This is a python demo page</title>
>>> tag = soup.a #获得第一个a标签内容，任何标签都可以通过soup.tag方式获得，存在多个时，返回第一个
>>> tag
<a class="py1" href="http://www.icourse163.org/course/BIT-268001" id="link1">Basic Python</a>

Name 标签名字

>>> soup.a.name #soup标签名字
'a'
>>> soup.a.parent.name #获得其父类名字
'p'
>>> soup.a.parent.parent.name
'body'

Attributes 标签的属性

>>> tag = soup.a
>>> tag.attrs #获取标签属性，若没有则返回空字典
{'href': 'http://www.icourse163.org/course/BIT-268001', 'class': ['py1'], 'id': 'link1'}
>>> tag.attrs['class'] #获取属性内容列表
['py1']
>>> tag.attrs['href']
'http://www.icourse163.org/course/BIT-268001'
>>> type(tag.attrs['class']) #标签属性类型
<class 'list'>
>>> type(tag.attrs)
<class 'dict'>
>>> type(tag)#标签类型，特殊定义
<class 'bs4.element.Tag'>

NavigableString 标签内非属性字符串

>>> tag = soup.a 
>>> soup.a.string #标签内非属性字符串
'Basic Python'
>>> soup.p
<p class="title"><b>The demo python introduces several python courses.</b></p>
>>> soup.p.string #标签内非属性字符串，可跨越多层次
'The demo python introduces several python courses.'
>>> type(soup.p.string)
<class 'bs4.element.NavigableString'>

Comment 标签内字符串的注释部分

#当获取非属性字符串时，注释和非注释需要通过其类型分辨
>>> newsoup =  BeautifulSoup("<b><!--Comment--></b><p>not comment</p>","html.parser")
>>> newsoup.b.string
'Comment'
>>> type(newsoup.b.string)
<class 'bs4.element.Comment'>  #注释类型
>>> newsoup.p.string
'not comment'
>>> type(newsoup.p.string)
<class 'bs4.element.NavigableString'> #非属性字符串类型

属性 说明
.contents 将所有儿子节点存入列表，返回列表类型

for child in soup.body.children: #遍历儿子节点

for child in soup.body.descendants: #遍历子孙节点

平行遍历

属性 说明
.next_sibling 返回按照HTML文本顺序的下一个平行节点标签
.previous_sibling 返回按照HTML文本顺序的上一个平行节点标签
.next_siblings 迭代类型，返回按照HTML文本顺序的后续所有平行节点标签
.previous_siblings 迭代类型，返回按照HTML文本顺序的前续所有平行节点标签

必须要在同一个父节点下的节点，才能进行平行遍历

>>> import requests
>>> from bs4 import BeautifulSoup
>>> r = requests.get("http://python123.io/ws/demo.html")
>>> demo = r.text
>>> demo
'<html><head><title>This is a python demo page</title></head>\r\n<body>\r\n<p class="title"><b>The demo python introduces several python courses.</b></p>\r\n<p class="course">Python is a wonderful general-purpose programming language. You can learn Python from novice to professional by tracking the following courses:\r\n<a href="http://www.icourse163.org/course/BIT-268001" class="py1" id="link1">Basic Python</a> and <a href="http://www.icourse163.org/course/BIT-1001870001" class="py2" id="link2">Advanced Python</a>.</p>\r\n</body></html>'
>>> soup = BeautifulSoup(demo,"html.parser") #将demo做成汤

#在标签之间的NavigableString，即标签内非属性字符串，也构成了节点
>>> soup.a.next_sibling #任意节点的平行节点使存在NavigableString的
' and '
>>> soup.a.next_sibling
' and '
>>> soup.a.next_sibling.next_sibling
<a class="py2" href="http://www.icourse163.org/course/BIT-1001870001" id="link2">Advanced Python</a>
>>> soup.a.previous_sibling
'Python is a wonderful general-purpose programming language. You can learn Python from novice to professional by tracking the following courses:\r\n'
>>> soup.a.previous_sibling.previous_sibling
>>>

基础结构

for sibling in soup.a.next_sibling: #遍历后续节点
    print(sibling)
for sibling in soup.a.previous_sibling: #遍历前续节点
    print(sibling)

bs4库的编码

bs4库将任何HTML输入都变成utf‐8编码
Python 3.x默认支持编码是utf‐8,解析无障碍
